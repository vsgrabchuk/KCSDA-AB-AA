# KCSDA-AB-AA

## Задача
Есть данные А/А-теста с '2024-02-23' по '2024-02-29'. Оценить корректность работы системы сплитования

## Общие выводы
- распределение p-value равномерное
- FPR ~ 0.05
- => система сплитования работает корректно
